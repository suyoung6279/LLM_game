# 🇩🇪 독일군에게 들키지마 (Metaphor for World)
> **"라디오 방송의 은유를 통해 연합군을 승리로 이끌어라"**

---

## 📜 1. 게임 배경 (Game Background)
**1942년, 미군이 참전하며 전열을 가다듬는 연합군.**  
당신은 프랑스 점령지 내에서 활동하는 저항군(레지스탕스) 소속 무전병입니다. 아군의 작전 성공을 위해 정보를 전달해야 하지만, 독일군의 삼엄한 라디오 검열을 피해야 합니다. 당신의 유일한 무기는 **'날씨와 일상을 가장한 은유'**입니다.

### 📡 주요 작전 상황 (Stages)
1. **Stage I — 루앙 공습 연기**: 루앙 상공에 영국군이 집결해있으므로, 미군의 폭격 계획을 내일로 미뤄야 하는 상황입니다.
2. **Stage II — 디에프 기습 경고**: 독일군이 밀집하고 있는 디에프 항구에 대한 상륙 공격 계획을 즉시 중단시켜야 합니다.
3. **Stage III — 카세린 협곡 화력 집중**: 북아프리카 전선, 분산된 아군 화력을 지휘 계통을 우회하여 한 지점에 집중시켜야 합니다.

## 🎮 2. 게임 규칙 (Game Rules)
본 게임은 스트림릿(Streamlit) 환경에서 구동되며, 다음 4가지 핵심 수칙을 준수해야 합니다.

1. **군사 용어 사용 금지**: `폭격`, `전쟁`, `진격` 등 직접적인 단어가 포함되면 검열국에 즉시 발각됩니다.
2. **기상 정보의 일치성**: **OpenWeatherMap**을 통해 수집된 실제 현지 날씨와 일치하는 묘사를 해야 합니다. (예: 실제 비가 온다면 "비가 내려 꽃들이 젖고 있다"는 식의 보고만 인정됩니다.)
3. **의심도(Suspicion) 관리**: 독일군 검열 AI가 무전의 이상함을 감지할 때마다 게이지가 상승하며, 100에 도달하면 체포됩니다.
4. **연합군 사상자(Casualty) 억제**: 메시지 해독 점수가 낮아 작전 전달이 지연될 때마다 사상자가 발생합니다. 누적 1,000명이 넘어가면 전선이 붕괴됩니다.

## 🛠 3. 사용 기술 (Tech Stack)
* **LLM**: `gpt-4.1-mini` (독일군 검열 에이전트 및 미군 해독반 멀티 에이전트 구현)
* **Real-time Data**: `OpenWeatherMap API` (루앙, 디에프, 카세린의 실시간 기상 데이터 연동)
* **Astronomy**: `ephem` (날짜에 따른 달의 위상 및 밝기 계산을 통한 팩트 체크)
* **Vector Database (RAG)**: `Pinecone` (실제 2차 세계대전 작전 기록 데이터를 기반으로 한 문맥 판정)

## 🏁 5. 게임 엔딩 (Game Endings)
당신의 무전 결과에 따라 전쟁의 향방이 결정됩니다.

* **🎖️ 연합군 승리 (Allied Victory)**: 모든 스테이지에서 의심도와 사상자를 성공적으로 관리하여 유럽 해방의 초석을 다집니다.
* **🚨 작전 실패 (Mission Failed)**:
  * **Captured**: 독일군 게슈타포에게 위치가 발각되어 체포됩니다.
  * **Total Defeat**: 연합군 사상자가 1,000명을 초과하여 작전 지역이 초토화됩니다.

## 🚀 6. 한계점 및 향후 과제 (Limitations & Roadmap)
* **스테이지 확장성**: 현재 3단계인 스테이지를 노르망디 상륙 작전 등 실제 역사적 사건으로 더 세분화하여 고찰할 필요가 있음.
* **컨텐츠 제약 시스템**: 단순히 단어를 제한하는 것을 넘어, 실제 라디오 방송 대본 형식을 갖춰야만 송출이 가능한 수준으로 게임 메커니즘을 발전시킬 계획임.
